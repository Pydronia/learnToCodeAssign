Tables

Let's now try adding a table to your website! Some basic tags you need to know include:

  • <table>: This tag essentially tells the browser that everything inside of it is part of a table
  • <tr>: The amount of these in a table is the amount of rows in a table - inside each of these you will have an amount of...
  • <td> tags: these are basically each of the cells that you want in your table - e.g columns.
  • <th>: These tags are usually used in the first <tr> or as the first cell in each <tr>, because they are table headings. By default, the browser will make any content in them bold and center aligned, as opposed to the standard left aligned and regular text in <td>'s

Try this on your own website! When you've finished, continue on reading.

There are also 5 attributes for tables that are useful to know. These are:

  • border: This will make each cell of the table have a border - this can be very useful to make sure you can see the sizes of each cell. Set it to "1" to make borders appear. Add this attribute onto your main <table> tag.
  • colspan: This tells the browser how many columns that specific cell will span - eg the cell will take up 2 columns instead of 1, if you set it to "2".
  • rowspan: Similar to colspan, this indicates how many rows the cell specified should span. Add colspan and rowspan attributes to <td>'s or <th>'s.
  • width and height: These attributes are rather self-explanatory - respectively, they will set the width or height of a block element - they won't work properly on most inline elements, as they don't have a size like block elements do. You can specify sizes in pixels, like 200px, as a percentage of the width of the users screen like 80%, plus a few units of measurement, which are not important to know right now.

Now try yourself - make this table in your webpage! Scroll down for the solution once you have done. Make the width of your table to be 500px and the height to be 150px.

Solution

<table border="1">
    <tr>
        <td colspan="2"></td>
        <td></td>
        <td></td>
    </tr>
    <tr>
        <td></td>
        <td></td>
        <td colspan="2"></td>
    </tr>
    <tr>
        <td rowspan="2"></td>
        <td></td>
        <td></td>
        <td></td>
    </tr>
    <tr>
        <td></td>
        <td></td>
        <td></td>
    </tr>
</table>

Note that the borders around your table may not be the same as shown above. Good job for finishing this exercise! One thing to note about this code that you have to get rid of any extra <td> tags so that the previous <td>'s can take their place, otherwise they'll be pushed out of the way, making a messy table!
