Creating your first web page

<!DOCTYPE html>
<html>
<head>
    <title>My first webpage</title>
</head>
<body>
    <p>
       Hello! This is my first webpage!
    </p>
</body>
</html>

Above, you can see an example of a fully functioning web page - let's go through the features of it before letting you make your own!

Things inside angular brackets (<>) are called tags - or elements. They tell the browser how to structure your content. There are many different tags that affect the structure of your content: some examples follow.

An example of an opening paragraph element tag would be <p>, and an example of a closing tag would be </p>. Using a <p> tag is the most common way to put text into your website.

Other tags that affect text include:

  • <em>: Tells the browser to emphasise the text. Most browsers simply make it italic.
  • <strong>: Tells the browser to make the text strong, usually making it bold.
  • <h1>: This defines a heading - there are 6 types, <h1>, <h2>, <h3>, and so on, all the way to <h6>. <h1> is the biggest heading, and <h6> the smallest.

Now, what is that <!DOCTYPE html> tag at the top? This is a tag that doesn't need to be closed (that is, have another tag with a slash in front of it) that tells the browser what type of document the browser is looking at - in this case it's html, so the browser will show the web page like it should, as HTML.

What about those <html>, <head>, and <body> tags? Put simply, you always have to put the <html> tag around everything in your web page. We always need it, as it represents the root of the HTML document. The <head> tag is where you put the title of your web page plus other information about it, as well as linking CSS stylesheets to it (which we will get to in later lessons)! The <body> tag is essentially where you put all the content you want to see on the screen - you should not put any content outside of it!

Now, try making your own web site! Use the code above as reference, however make it so it has a title that says "Hello, World!", a <h1> heading that says "My first website!" and a <p> paragraph that also says "Hello, World!" Check your website by going to the address of the new HTML document you have made, and then check your code with the solution below when you have done!

Hint: Right click in the directory tree to the left of ShiftEdit and create a new HTML document to start, then go to cgscomputing.com/studentid/name_of_html_document.html to see your website. If you name your file "index.html", your browser will automatically show that page if you go to cgscomputing.com/studentid/.

Solution

<!DOCTYPE html>
<html>
<head>
    <title>Hello, World!</title>
</head>
<body>
    <h1>My first website!</h1>
    <p>Hello, World!</p>
</body>
</html>

If you got this, or something similar to this, good job! In HTML, spaces and new lines outside of tags don't matter at all - however we use indentation to make it readable for other fellow coders. This involves using tabs on tags and content that are nested (inside) another tag. The indentation above is a good idea of how your code should look.
