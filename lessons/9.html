Introduction to CSS

Cascading Style Sheet (CSS) is the name for the coding language used to add visuals and style to your HTML, with color, graphics, animations, positioning, and more. CSS was introduced in December 17, 1996 and was developed by HÃ¥kon Wium Lie, Bert Bos and the World Wide Web Consortium. Being a separate language, it keeps style (CSS) and structure (HTML) separate, which makes it easy to make distinctions between the actual meaning of a document, and how that meaning is presented to your viewers. This is also good as it assists with machine readability, or how easily the browser can interpret your document. Below is an example of some CSS, and its structure:

p {
    color: red;
}

selector {
    property: value;
}

Let's break this down. The selector is what we actually want to style - it could be an element, a class, or an ID, of which the latter two we will learn about later. So, if we want to style all the <h2>'s in our HTML, we would use "h2" as our selector. After the selector we must have our properties and values inside curly braces ({}). A property is the specific aspect about the selector that we want to change - it could be the text color, the size of the font, the type of font, or much much more. After the property we have a colon (:) and the value - this is what we actually want the property to be set to, like which text color, or what font size. This statement always ends in a semicolon (;).

Above is essentially what CSS is in a nutshell. In the next lesson we will learn how to actually apply this to our HTML.
