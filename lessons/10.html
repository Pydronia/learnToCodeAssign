Your first CSS styles

Now, let's start styling the web page you currently have! There are 3 ways of styling a HTML document: inline, internal and external. To show the differences, let's make the paragraph in our website that we already have's font colour to be red. To do this, we use the property color. This simply defines the font's color, and we can do this using a hexadecimal value, an RGB value, or one of the already built in colours that CSS provides for us, like blue or pink (see https://en.wikipedia.org/wiki/Web_colors for more). Note that all CSS properties are spelt using the American spelling, so colour (with a "u") will not work.

The inline way to style elements is as follows:

<p style="color: rgb(255, 0, 0);">Hello, World!</p>

We use the style attribute in our HTML element to directly affect just that tag without using a selector. In this case, I have used the RGB style of a pure red colour. This way of styling is easy and simple, though it can get messy easily and it makes it difficult to use multiple properties. For internal styling, we use the <style> tag inside the <head> of our HTML:

<style>
    p {
        color: #FF0000;
    }
</style>

In this case, the CSS is in the same file as all our HTML. I've used the hexadecimal style of indicating a pure red colour. This method makes it easier to read our CSS and add multiple styles, however both style and structure is in the same document, which is not the best way to go about doing things, especially when working with multiple documents, as you will have to copy and past the same style in all your different HTML documents! External styling is the solution to this. All we do is link a CSS file to our HTML using, coincidentally, a <link> tag. This means we can use the same CSS file with multiple HTML files with ease, and further separate our style from our structure:

In our HTML we have:

<link href="path_to_css_file" rel="stylesheet" type="text/css" />

href tells where the CSS file is, rel tells the browser how the linked file relates to the HTML, and type tells the browser what kind of file is linked.

Create a CSS file (with the extension .css) somewhere, but remember to keep your project organised! Our recommendation is to create the style sheet inside a folder called "css" or similar in your root directory. Add the link tag like above, except with the correct href value, and put the following code in your newly created CSS file:

p {
    color: red;
}

And reload your page! If you have successfully done this, your "Hello, World!" paragraph should be red! Well done - that was your first taste into CSS styling.
