Common CSS styles

Now that you have a basic understanding of how CSS styling works, here is a list of many different CSS properties that are extremely useful to know - if you want to learn even more, head over to W3 School's CSS Reference for a much more comprehensive list.

It's also useful to know what units you can use when setting the value of properties. You can use pixels (px), a percentage of the height or width (%), and em, which is the default font-size of the users device, so 2em would be 2 times the default font size. There are many more units, however they are the most commonly used and all you will need to know for now.

  • color: sets the font color of the selected element(s)
  • font-size: sets the size of any font inside the selected element(s)
  • background-color: sets what colour the background is behind the content of the selected element(s)
  • border: creates a border around the selected element(s), see http://www.w3schools.com/css/css_border.asp for more.
  • width, and height: sets the width or the height respectively of the selected element(s)
  • margin: sets a margin around the selected element(s), generating space around the outside of the element. Margins create space outside a element's border. See http://www.w3schools.com/css/css_margin.asp for more.
  • padding: sets padding around the content of an element, generating space inside a element's border, pushing it inwards and towards the center of the element. See http://www.w3schools.com/css/css_padding.asp for more.
  • display: This property will set how the element is displayed - for example, you can get the selected element(s) to display as if they were a block element, even if they were inline elements. See http://www.w3schools.com/css/css_display_visibility.asp for more.
  • text-align: sets how text is aligned inside the block element - eg setting this property to center will cause text to be centered in the selected element - like using the formatting options in Microsoft Word.
  • font-family: sets, in priority, what font the selected element's should use. See http://www.w3schools.com/css/css_font.asp for more.
