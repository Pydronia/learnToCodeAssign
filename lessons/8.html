Div's and Spans

Before we enter the CSS portion of the course, it would be good to learn about 2 other HTML tags that will be used extensively throughout your time developing for web.

Every single element you have used in HTML is either a block element, or an inline element. Block elements are arranged down the page - e.g. 2 block elements, one after each other, will be arranged one below the other. Essentially, they always start on a new line. Examples of these are <p> paragraph elements, <h1> to <h6> headings, and tables. Inline elements usually affect text inside block elements - they might change the style of some text, and don't have to begin on a new line. Examples of inline elements include <em>, <strong>, and images. One main point to keep in mind is that block elements can have both block elements and inline elements inside them, but inline elements can only have other inline elements in them - it's bad practice to have a block element inside a inline element.

The <div> element is a dummy block element. This means it applies no styling to content inside whatsoever by default. It is simply a way to break up an entire section of HTML into divisions, and to style those specific divisions using classes and ID's (which is a part of CSS, to be introduced in later lessons. This is the same with the <span> element. This is a dummy inline element, as compared to the block element <div>. It does nothing but provide structure to the web page and a means of styling individual parts of text.

This concludes the HTML portion of the course. In the next lesson, you will begin to learn CSS and how to style <div>'s and <span>'s properly to make beautiful looking websites!
