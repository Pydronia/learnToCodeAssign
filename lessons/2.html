Setting up your IDE

Now, we can write code using any old text editor, but it's very hard to write readable code - instead we use what's called an IDE, which stands for Integrated Development Environment. It's essentially a fancy text editor with colors that help your write code - in this case we will be setting up ShiftEdit.

First, go to http://www.shiftedit.net. Create a new account using your school email, and check your email for a confirmation link. Once you've logged in successfully, you should be presented with this screen. Click the hamburger, which is an icon in the top left that looks like three lines on top of each other. Then click "New Site". Upon clicking this the ‘Site Settings Window’ will open. Fill in the following forms with;

  • Name: cgscomputing.com
  • Server Type: FTP
  • Domain: ftp.cgscomputing.com
  • Port: 21
  • Username: studentid@cgscomputing.com (e.g. 1234@cgscomputing.com)
  • Password: Your birthday in DDMMYY format (e.g. 041198 is 4 November 1998)
  • Save: Ticked
  • Dir: leave blank
  • Web URL: http://www.cgscomputing.com/studentid (this must be all lowercase)

Now you're all set up and ready to create your first website!
